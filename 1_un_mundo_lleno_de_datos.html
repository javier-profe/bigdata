1. Un mundo lleno de datos

1. Los datos son importantes

Seguro que no eres consciente de que navegas diariamente entre grandes volúmenes de datos que se almacenaran, se clasificaran y se analizaran. A partir de estos datos se obtiene conocimiento que se utiliza para ofrecer productos y servicios. La ciencia de datos o el fenómeno big data como hoy lo conocemos, está revolucionando la ciencia, la economía, la política e incluso nuestro estilo de vida.

A partir del análisis de estos datos puedes obtener un conocimiento muy valioso. Vamos a conocer más de este mundo de los datos masivos y cómo afectan ya a tu vida.

El objetivo de este reto es conseguir ser capaz de obtener datos un conjunto de datos desde algún origen conocido sobre algún fenómeno social, económico, etc. Además, hacer un análisis y representación con alguna herramienta de big data  y finalmente hacer una reflexión final de acuerdo al análisis de esos datos.

2. Datos por todas partes

El big data se refiere a conjuntos de datos muy grandes y complejos que son demasiado voluminosos y complejos para ser procesados y analizados de manera efectiva mediante técnicas y herramientas tradicionales. Estos conjuntos de datos pueden incluir una amplia variedad de información, como registros de transacciones comerciales, datos de sensores, imágenes y videos, información de redes sociales y mucho más.

3. ¿Qué vas a aprender en este reto?

En este reto conocerás muchos conceptos nuevos relacionados con el big data y profundizarás en los siguientes conocimientos:

    • Ciencias de datos y simulaciones.
    • Big data. Características.
    • Volumen de datos.
    • Visualización, transporte y almacenaje de los datos.
    • Recogida, análisis y generación de datos.
    • Simulación de fenómenos naturales y sociales.
    • Descripción del modelo.
